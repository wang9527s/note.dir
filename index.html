Mon, Jan 1, 0001  突然想弄一个blog网站玩一玩。 最后选择了静态网站+github page的方式，简单实用。 和hexo相比 hugo更好用，并且初始版本中没有太多的文件。 1. hugo环境搭建 sudo apt install hugo # ubuntu源中的版本有点低，后来在官网中下载安装 hugo new site hugo-blog cd hugo-blog/themes git clone https://github.com/spf13/hyde.git cd .. hugo new blog环境搭建.md hugo server --theme=hyde --buildDrafts 然后localhost:1313就可以在浏览器中看到blog的首页了。 2. github page 在git上建立一个aaa.github.io的仓库，aaa是自己的用户名称。
