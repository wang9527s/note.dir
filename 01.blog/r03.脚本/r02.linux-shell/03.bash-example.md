
### 查找包含关键字的文件路径

```bash
find . -type f -name "*" -exec grep -l "$keyword" {} \;
```

### 对文件夹中指定类型的文件，使用egrep

```bash
find . -type f -name "*.txt" -exec egrep -C2 "pthread" {} +
```

### 对grep的结果进行处理

```bash
# 只显示冒号之后的内容
| sed 's/^[^:]*://' 
# 移除冒号和空格
| sed 's/[: ]//g'
# 排序并去重
| sort | uniq

# 根据指定字符分割，显示第一部分
| cut -d ":" -f 1 
# 根据字符串分割
| awk -F "device" '{print $2}' 

```
